Опис роботи

Даний репозиторій містить виконану лабораторну роботу №1 з дисципліни
«Крос-платформене програмування»

У роботі продемонстровано практичне використання механізму Reflection в Java
для аналізу структури класів, дослідження об’єктів та динамічного виклику методів



Мета роботи

Метою роботи є:

- ознайомлення з Java Reflection API
- аналіз структури класів під час виконання програми
- дослідження стану об’єктів
- динамічний виклик методів
- створення масивів під час виконання програми
- використання механізму динамічних проксі



Структура проєкту

Проєкт складається з наступних пакетів:

- `lab1.task1` – Аналіз структури класу (консольна та GUI версія)
- `lab1.task2` – Аналіз об’єкта та виклик методів
- `lab1.task3` – Динамічний виклик методів за ім’ям
- `lab1.task4` – Створення та обробка масивів через Reflection
- `lab1.task5` – Динамічні проксі та профілювання викликів



Використані технології

- IntelliJ IDEA
- Git
- GitHub



Запуск програми

Для перевірки роботи необхідно запускати відповідні класи:

- `MainTask1`
- `MainTask2`
- `MainTask3`
- `MainTask4`
- `MainTask5`

Кожен клас демонструє виконання окремого завдання лабораторної роботи
